# Vega Cards

An experimental prototype front-end for interactively rendering visualisations in Vega/Vega-Lite. A simple composition of [reframe](https://github.com/day8/re-frame), [reagent](http://reagent-project.github.io) and [hanami](https://github.com/jsa-aerial/hanami) with [material-ui](https://github.com/jsa-aerial/hanami) styling.

# Installation

Requires Shadow-CLJS. Run using `shadow-cljs watch app` and navigate to `localhost:8020`.
